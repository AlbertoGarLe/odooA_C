1. Búsqueda de plantilla

Aquí describiremos cómo seleccionamos y evaluamos diferentes plantillas para este proyecto.

Realizamos una búsqueda en la red virtual y hallamos numerosas plantillas, de las cuales, las únicas que nos llamaban la atención eran las minimalistas.



2. Istalación

La instalacion de una plantilla en nuestro equipo es sencilla, solo tenemos que descargar la plantilla que tengamos y ubicar su archivo index junto con su css



3. Estructuración

Tenemos que buscar una manera de organizar nuestra pagina, luego insertaremos el contenido modificando el que ya nos proporciona la plantilla segun lo que nosotros vamos necesitando



4. Modificación

En este apartado explicaremos los cambios realizados para personalizar la plantilla seleccionada.

Importante!
